# MergeSortProject-DataStructuresAndAlgorithms

1. Write the stages of the above array according to the sort type.

```
0. [16,21,11,8,12,22]
1. [16,21,11] --- [8,12,22]
2. [16,21] - [11] --- [8,12] - [22]
3. [16] - [21] - [11] --- [8] - [12] - [22]
4. [16,21] - [11] --- [8,12] - [22]
4. [11,16,21] --- [8,12,22]
5. [8,11,12,16,21,22]
```

2. Write the Big-O notation.

